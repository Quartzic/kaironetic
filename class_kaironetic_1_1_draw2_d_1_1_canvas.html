Detailed Description

Represents a 2D canvas.

Canvas provides the appropriate facilities to draw on a plane; for example, a piece of paper. It will store a set of Draw2DCommands and sort their draw order to decrease unnecessary travel; therefore, do not rely on a specific ordering. If you attempt to draw outside of the defined boundaries of the Canvas, an exception will be thrown.
